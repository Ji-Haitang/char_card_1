# 钱塘君剧情事件完整大纲——"西风残照·伊州篇"（最终修订版）

## 总纲

### 核心剧情线
党项大军压境伊州 → 悬赏告示张贴 → 天山派协助守城 → 战事不利 → 钱塘君隐瞒所有人独自献身 → 佯装日常的永别 → 主角发现真相 → 党项退兵 → 天山派敢死队夜袭救人 → 主角闯入大帐与钱塘君对峙

### 核心悲剧逻辑
钱塘君隐瞒了所有人——包括洞庭君——独自行动。没有诀别信，没有任何直接的告别痕迹。她把离开伪装成了一次再普通不过的外出，让所有人以为她很快就会回来。真相是事后通过碎片拼凑才逐渐浮出水面的。

### 母亲真相
钱塘君的母亲被俘后，实际上被西夏太子（钱塘君生父）礼遇相待，二人真心相爱，最终成婚并生下钱塘君。太子主张和平、爱好汉文化，因此遭二皇子阴谋陷害被废，抄家灭门。**钱塘君知道父母相爱的真相**，但天山派女弟子嫁给西夏太子一事有辱门派名声，因此天山派对外宣称其母"被俘受辱"。钱塘君被迫接受这个谎言作为自己的"官方身世"，在所有人面前扮演"耻辱之女"。她真正的痛苦不是不知道真相，而是知道真相却不能说——她必须亲手否认父母的爱情，亲手将自己的出生定义为耻辱，以维护天山派的体面。

### 好感度预设
40-60区间（暧昧未明，爱而不自知），以最大化"错过"的悲剧张力。

## 第一幕：烽烟起——"那张画像上的小女孩"

### 场景一：斥候截获（天山外堡）

天山派外堡巡逻弟子截获一队党项斥候，搜出军令文书和悬赏告示。告示上画着一个七八岁的党项女孩——银白头发，额侧有角状突起——悬赏金额高得离谱，备注"李氏宗女，活捉万金"。长老会紧急召开。破阵子不明就里，拍桌质问为何三万铁鹞子只为找一个小丫头。玄天青与洞庭君对视一瞬，洞庭君茶杯微颤，苓雪妃闭眼摩挲剑柄。玄天青将告示翻面朝下："先议守城。此事容后再议。"全程主角不在场——他在外堡帮难民修缮房屋。钱塘君也不在场——她在后山秘密基地躺在石头上晒太阳，龙尾懒洋洋地晃荡。这是她最后一段真正无忧无虑的时光。知情者的沉默与不知情者的日常，从第一幕就拉开信息差。

### 场景二：告示流入伊州（伊州城门）

天山派派弟子下山协助伊州守军加固城防。主角与钱塘君等人同行。城墙上已张贴党项悬赏告示。主角在城门口随口对呼延显说："这画像上的小女孩，眉眼倒有几分像钱塘小时候。"呼延显脚步一顿，随即用玩世不恭的语气岔开话题，不动声色挡在主角与告示之间。钱塘君走在队伍最后，经过告示时脚步未停、表情未变，甚至还在和萧白瑚斗嘴——但玩家可以注意到她的龙尾在看到告示的瞬间猛地绷直，随即恢复正常晃荡。她追上主角，一巴掌拍在他后脑勺："磨蹭什么！城里新开了烤馕店，晚了就没了！哇哈哈哈！"所有知情者都在替她隐瞒，而她自己也在演——但她演的不只是"我不是画像上的人"，更是"我是被俘受辱生下的孩子"这个她被迫背负了十九年的谎言。

### 场景三：深夜的悬崖（天山后山·秘密基地）

当夜，钱塘君独自来到后山悬崖边。她坐在崖边，手里攥着母亲留下的玉佩，玉佩背面刻着党项文字。月光下她翻来覆去地看，龙尾无意识地缠绕石柱越缠越紧。独白中她计算着三万铁鹞子的兵力，想到外堡的难民和孩子，想到如果党项人知道她在天山派会怎样。她低声说："爹爹，娘亲……你们当年想要的和平，到头来什么都没换到。"她知道父母相爱，知道父亲是因为主张和平才被废被杀，但这些她不能对任何人说。在天山派的叙事里，她的母亲是"受辱而死"，她的父亲是"党项蛮贼"。她必须活在这个谎言里，因为真相会让天山派蒙羞。她仰头看星，轻轻笑了一声——不是"哇哈哈哈"，是像叹息一样的笑。"……我连替你们喊冤都做不到。"这种"知道真相却被迫沉默"的痛苦，比无知更加腐蚀人心。

## 第二幕：城围——"本龙去帮你们打仗啦"

### 场景四：天山派守城（伊州城墙）

党项大军抵达伊州城外，铁鹞子重骑列阵，旌旗遮天。天山派全派出动协助守城。钱塘君在城墙上表现异常亢奋，挥舞"开山"大杀四方，每劈翻一个攀城士兵都要大笑："哇哈哈哈！买一送一！"战斗力惊人，城墙东段固若金汤。但玩家会注意到主角注意不到的细节：她听得懂党项士兵的西夏语喊叫——有人喊"冲"，有人喊"娘"，有人喊"救命"；她的斧法越来越狠，像在用暴力压制某种情绪；一个不超过十五岁的党项小兵攀上城墙，她举斧顿了一瞬，用斧背将他拍下而非劈死。战后她独坐城墙垛口，看着党项军营万千灯火，独白："他们也是爹爹曾经想要保护的子民。"她杀的不是"仇人"，而是父亲曾经想要带向和平的族人。这种身份撕裂比单纯的"敌我矛盾"更加深刻。

### 场景五：使者通牒（伊州守将府 / 长老会）

围城第五日，党项军切断水源，守军伤亡过半。党项使者入城递交文书，只有一句话："交出李氏宗女，大军即刻退兵。否则，屠城。"长老会秘密召开。破阵子拍桌："死在城墙上也不交人！"玄天青冷静陈述城内两万百姓的处境。洞庭君沉默后开口："不能让钱塘知道。"苓雪妃靠在墙边闭眼："她已经知道了。今天在城墙上她放过了一个党项小兵。钱塘的斧法从不留手。"洞庭君龙尾猛地绷直。玄天青下令加强监视。而此刻主角只知道长老们开了次会，气氛凝重。他问钱塘君怎么回事，钱塘君一脸无所谓地塞给他一块加了十八种药材的肉干，主角吃完当场喷鼻血，她笑得前仰后合。玩家刚看完剑拔弩张的长老会，紧接着就看到她笑嘻嘻塞肉干——她什么都知道了，但她还在笑。

### 场景六：钱塘君的暗中行动（伊州暗巷 / 党项军营外围）

围城第七日。钱塘君以"巡逻"为由独自行动：销毁城内与自己身份相关的线索，利用精通西夏语的优势夜间潜近党项军营偷听情报，寻找能独自接近党项主将的路线。她趴在沙丘后听到军官对话：将军只要拿到李氏宗女就撤兵，对伊州城本身没有兴趣。确认了"只要自己出去，围城就会解除"这个事实后，她仰面朝天，用西夏语轻声说了一句游戏不提供翻译的话——"额布格，额涅……孩儿不孝。"（父亲，母亲……孩儿不孝。）回城时她路过伤兵救治点，看到主角正温柔地照顾体力透支的鹿椿若——替她倒水、整理药材。她站在门外看了几秒，无声地笑了一下，转身走了。独白："你对谁都这么好。所以对我好的时候……也没什么特别的吧。"这个错误判断成为她决定独自赴死的心理推力——她觉得自己走了，他也不会太难过。

## 第三幕：诀别——"本龙去买个东西，很快回来"

### 场景七：最后的日常（围城第九日·天山后山）

围城第九日，党项军发出最后通牒：明日正午前交人，否则屠城。钱塘君向洞庭君请假，说要带主角回后山"放松一下"。洞庭君看了她很久，点了头——她不知道这是妹妹的告别。这一天是整条剧情线中最温暖的日常：两人烤鱼（糊了）、扔石子（她赢）、她教主角说"外族话"——"塔西·德力格尔"，她说意思是"今天天气真好"，实际是西夏语"请你平安"。阳光下她忽然问："如果有一天本龙不在了，你会不会想我啊？"主角没多想："你能去哪？难不成你还能飞了？"她哇哈哈大笑："说得也是！本龙哪儿也不去！"笑着，背后的手攥成拳头，指节发白。回程时她在夕阳中回头看主角，露出一种他从未见过的神情，然后笑着说："今天挺好的。"每一个日常细节都是伪装成日常的告别，但没有任何一个人知道——包括洞庭君。

### 场景八：天亮前的房间（围城第九日深夜·主角房间）

深夜。钱塘君来到主角房间。主角睡得正沉。她站在床边看了很久，蹲下来近距离看着他的脸，独白中说出了"塔西·德力格尔"的真实含义，说"骗你的，本龙最爱骗你了"。她伸手想摸他的脸，手指悬在半空，最终缩回——"不碰了。碰了就走不了了。"她从怀里掏出玉佩想留下，犹豫再三又揣回去——"留了你会去查，查了你就知道了。你来找我你就会死。"她走到门口回头，极轻地说："本来想叫你一声名字的。但是叫了，我怕我走不了。"然后："再见。哥们儿。"她没有留下任何信件、纸条或物品。她把"开山"留在自己房间原位，一切如常，不留痕迹。她要让所有人以为她只是出去转转，很快就会回来。赤手空拳，她走进了伊州城的夜色里。走出天山派的那一刻，她回头望了一眼雪山，用只有自己能听到的声音说了一句西夏语——那是她父亲教给母亲的一句话，意思是"我会回来的"。但她知道自己不会了。

## 第四幕：发现与营救——"她去哪了"

### 场景九：清晨的失踪（围城第十日·拂晓）

天亮。主角被号角声惊醒，匆忙赶往城墙集合。他没有注意到任何异常——钱塘君的房间门关着，"开山"还靠在墙边，一切看起来都很正常。他以为她还在赖床，甚至笑骂了一句"这疯婆娘又赖床"。直到集合点位时，呼延显问了一句："钱塘呢？"没有人回答。主角去敲她的门，没人应。推开门——床铺整齐得不像话，像是被刻意整理过。但"开山"还在，衣物还在，看起来不像是离开。洞庭君赶到，看到整齐的床铺，脸色骤变。她翻遍房间，发现钱塘君母亲留下的玉佩不在了——那是她唯一带走的东西。洞庭君的手开始发抖。"她把玉佩带走了。"这句话让在场的知情者全部沉默。主角不明白一块玉佩意味着什么，但他从所有人的反应中感到了一种无法言说的恐惧。

### 场景十：碎片拼图（围城第十日·上午）

主角开始疯狂寻找钱塘君。他跑遍她所有常去的地方：后山秘密基地、偷看女澡堂的山崖、藏零食的树洞。每到一处都空无一人。与此同时，玄天青紧急排查城门记录，发现寅时有一人独自出城，守门兵描述"银白长发的姑娘，说是奉长老之命出城侦察"。玄天青闭上眼，脸颊上的白色蛇鳞在晨光中泛着冷光。洞庭君听到消息后没有任何表情，只是转身走进自己的房间，关上了门。门板后面传来一声极轻的、被死死压住的哽咽——她是钱塘君的姐姐，是最亲的人，但钱塘君连她都瞒了。这比被告知更痛——被信任的人排除在告别之外，意味着对方连让你痛苦的机会都不想给你。主角仍然不知道全部真相。他只知道钱塘君独自出了城，而城外是三万铁鹞子。他抓起钱塘君房间里的"开山"就要冲出城门，被破阵子一把按住。

### 场景十一：真相坍塌（围城第十日·玄天青处）

玄天青将悬赏告示摊在主角面前，告诉他全部真相：钱塘君是党项废太子的遗孤，三万大军围城只为找她。但他只说了天山派的"官方版本"——母亲被俘受辱，生下钱塘君后难产而死。他没有说出真相，因为他自己也不知道全部真相（或者知道但选择维护门派体面）。主角看着画像上银白头发、额侧龙角的小女孩，这一次他认出来了。然后所有碎片同时归位——她为什么那天那么开心，为什么教他说"外族话"，为什么问"如果本龙不在了"，为什么说"哪儿也不去"时手在背后攥成拳头。每一个碎片都是一把刀，此刻同时扎进来。他声音嘶哑："她什么时候走的。""寅时。约四个时辰前。"主角转身就走。玄天青厉声："你出城就是送死。"主角吼："那她一个人出去就不是送死吗？！"

### 场景十二：党项退兵（围城第十日·午后）

正午刚过。城墙上的守军发现异动——党项军阵开始收缩，大量辎重车队向西北方向移动。围城的压力骤然减轻。斥候带回消息：党项军已经抓到了目标，正在拔营撤退。主帅大帐向西北转移，预计天黑前全军撤离伊州外围。消息传回天山派，长老会陷入沉默。破阵子第一个拍桌站起来："追。"玄天青冷静分析：党项主力三万，天山派能战之人不足五十，正面追击无异于送死。但如果趁夜色、趁对方拔营混乱之际，组织一支精锐小队潜入——不求击溃大军，只求摸进主帅大帐，把人抢回来。破阵子看向苓雪妃。苓雪妃睁开眼，站起身，一言不发地走向兵器架取下长剑。洞庭君的声音很轻但很稳："我去。"主角攥着"开山"站在门口，眼眶通红："我也去。"没有人拦他。这一次，所有人都知道——这是一条有去无回的路。但没有人退。

但主帅大帐周围的守卫并未被完全调走。仍有二十余名精锐甲士环绕帐外，另有一名满头白发的党项老将坐镇——此人甲胄陈旧但一丝不苟，腰间佩刀磨得发亮，眼神沉稳如山
洞庭君观察片刻，低声对主角说："守帐的兵调不走。我去引开他们。你等我把人带走，再进去。"
主角："长老一个人——"
洞庭君打断他："半个时辰。不管发生什么，半个时辰后你必须离开。"她顿了顿，冰蓝色眼眸在火光中明灭不定。"……把她带回来。"
这句话和破阵子说的一模一样。但从洞庭君嘴里说出来，尾音有一丝几不可闻的颤抖
14-B｜洞庭君现身
洞庭君不再隐藏，直接从暗处走出，白蓝袍服在火光中格外醒目。她拔剑，一米四五的身体爆发出与体型完全不符的凌厉气势
她没有偷袭，而是正面现身——故意暴露自己。银白短发、青色龙角，在火光下辨识度极高
守帐甲士立刻警觉，为首的军官一声令下，二十余人朝洞庭君围拢
洞庭君且战且退，将守卫从大帐方向一步步引开。她的剑法凌厉精准，每一剑都只伤不杀——不是心慈手软，是不想制造尸体引来更多人。她要的是把这些人拖住，拖得越远越好
帐前守卫被引走大半，只剩那名白发老将和两名亲兵
14-C｜老将让路
主角从暗处冲出，制式长剑出鞘，直扑帐门
两名亲兵举矛拦截，主角以天山剑法连削带刺，三招之内将二人击倒
老将拔刀。刀光一闪，主角本能格挡——刀势极沉，震得他虎口发麻。这个老将的功夫远在那些甲士之上
但老将只出了一刀。
他看着主角，目光从他脸上扫过，停在他腰间佩戴的天山派弟子腰牌上。然后他的目光变了——不是敌意消退，而是一种更复杂的东西。像是确认了什么，又像是在做一个艰难的决定
老将收刀入鞘。侧身让开帐门
主角愣了一瞬。老将没有看他，只是用带着浓重口音的汉话说了一句："进去。"
主角没有时间犹豫。他冲进了大帐

### 场景十五：大帐对峙（主帅大帐内）

帐内灯火通明。主角看到的不是他认识的那个钱塘君。眼前的女人身穿党项郡主华服——赤金织锦长袍，凤翅金冠，腰系白玉带，衣摆拖地三尺。银白色长发按党项贵族样式高高盘起，脊背挺直如松，双手交叠放在膝上，目光平视前方。龙尾被华服长摆完全遮住，看不到任何晃动。她以"李钦若"的身份，用端庄疏离的语气，层层递进地编造谎言试图将主角推开：

**第一层——身份否定**：她自报本名"李钦若"，宣布自己是党项废太子之女、西夏宗室郡主，"钱塘君"只是天山派给的假名。语气如宣读诏书，端庄而不带感情。（闪回对比：幼年外堡，主角替被欺凌的她挡石子。）

**第二层——动机否定**：她声称自己在天山派十三年是为了替父亲收集河西情报，到处结交朋友只是为了套取消息，自己从第一天起就是间谍。语气平静如陈述事实，甚至带着淡淡怜悯。（闪回对比：童年天山，两人趴在悬崖边扔石子，她龙尾兴奋地勾住他手腕。）

**第三层——情感否定**：她说主角了解的只是她"演了十三年的角色"，拉他逃课是为了套情报，塞肉干是因为他好骗，把他按进雪堆是觉得他蠢。语气垂眸低缓，像先生纠正学生的错误。（闪回对比：姐妹相认之夜，她被带走前回头看着月光下的主角，他冲她挥手，她只是看了很久。）

**第四层——未来否定**：她说自己要回西夏做郡主，朝廷已答应恢复封号，绫罗绸缎、侍女成群才是她该有的生活，一把天山派的破斧头不配带走。语气淡漠如谈论一件早已安排好的行程。（闪回对比：数年后天山重逢，她从背后一巴掌拍在主角后脑勺上，龙尾甩得像风车，笑喊"你终于上来了！"）

**第五层——关系否定**：她以郡主身份俯视主角，逐条丈量二人之间的距离——她是正妃嫡女，他是天山派普通弟子，连长老都不是。"我是郡主，你是布衣。我们之间隔着的不是一座天山，是整个天下。你高攀不起。"语气无懈可击，姿态端庄如画。（闪回对比：快速剪辑——后厨塞肉干喷鼻血、后山逃课被破阵子追、雪地里互塞雪球满山跑、她靠在他门口抱着肉干睡着、秘密基地黄昏她把头靠在他肩上说"今天挺好的"。）

**裂缝与反击**：主角不用情感打动她，而是用"证据"逐条拆穿——放过小兵说明她不是间谍、肉干加药材说明她在乎他的身体、不带"开山"说明她不打算回来。最后他问出致命一击："那你为什么教我说'塔西·德力格尔'？那不是'今天天气真好'。那是'请你平安'。那是你在跟我道别。"钱塘君的伪装出现无法掩饰的裂痕——十指收紧、指节泛白、尾音微颤。她用尽全力维持住最后的平静："……你想多了。"

**兵刃相向**：主角将"开山"横在两人之间："你要是真的不想回去，拿起你的斧头，打赢我，我就走。"钱塘君看着那把她用了十三年的斧头，伸手握住斧柄。她的手在抖。她举起"开山"，摆出开山九式的起手式。在举斧的瞬间，所有伪装彻底崩塌——她没有用"李钦若"的声音，而是用回了"钱塘君"的声音，极轻地说："最后一次。让本龙……任性一回。"

场景十六：决斗（主帅大帐内）
16-A｜起手——两个人的开山九式
钱塘君持"开山"，标准起手式"劈岳"，沉默无声，与平日练功时的张扬判若两人
主角仅有一柄缴获的缺口制式长剑，第一式格挡即断
一至四式连贯劈出，帐内桌案屏风尽碎
关键细节：她的步法从第三式开始紊乱——呼吸节奏中夹着被压住的哽咽。她不是在打他，是在用暴力劈碎自己心里那个想跟他回去的念头
16-B｜破局——他用她教他的东西打赢了她
第五式"碎岩"劈下时，主角扔掉所有武器，切换为开山九式的步法
他从未正式学过开山九式。但他看了她练功十三年——每一个转身角度、每一次换步时机、每一式之间的呼吸节奏，不是刻意记的，是看她看了十三年自然就记住了
他空手以开山步法穿行于她的斧法之中：走她的步法 → 预判下一斧 → 贴着斧刃三寸闪避。不是在躲，是在她的斧法里游走
决胜一击：第八式"移山"与第九式"倒海"之间有一个极短的换气间隙（藏在呼吸节奏里，是钱塘君的个人习惯）。他数过她的呼吸。他在间隙中扣住她手腕，左手按上她握斧的手背，五指嵌入指缝——两人共同握住斧柄
这个姿势复刻了十三年前张天义教她握斧的场景。她有力气挣脱，但她没有动
关键台词：
主角："你的步法从第三式就乱了。你师父说过，心乱则步乱，步乱则斧偏。你每一斧都在避开我的要害。"
钱塘君（声音碎裂）："你什么时候……" / 主角："看你练了十三年。" / 钱塘君："……你这个笨蛋。你学什么不好……你学这个干什么……"
"开山"从两人手中滑落。主角看到她华服下摆露出的淬毒匕首
16-C｜匕首——她从一开始就没打算活着
匕首极小，刃泛幽蓝，淬天山梅花镖之毒，见血封喉
主角推断（信息差反转）：二皇子不在军中，在千里外的兴庆府。钱塘君的真实计划——以李氏宗女身份主动回到兴庆府面见二皇子，在所有人确认身份后近身刺杀，与仇人同归于尽。匕首够划一道伤口，毒够杀一个人，但她刺出去的同时也必死无疑
同时结清两笔账：父母血债（杀二皇子）+ 伊州活路（二皇子死则擒生军群龙无首，围城瓦解）
关键台词：主角（声音碎成渣）："你从一开始……就没打算活着回来。"
16-D｜崩塌——她终于哭了
心理防线逐层崩塌：
算账逻辑："我爹的账，我娘的账，天山派养我十三年的账，伊州两万条人命的账。我把他也带走，账就清了。多划算。"
身份撕裂："我在城墙上杀了他们。我听得懂他们喊什么。有人喊'额涅'，那是'娘'的意思。"
控诉世道："我知道不是我的错！但是有什么用？！错的是二皇子，错的是这个烂透了的世道——但他们不会受罚。最后买单的永远是我们这种人。"
隐秘心事："我怕你拦我。我怕你来找我。我怕你死。""你对谁都好……少一个钱塘君，你也……"（说不下去）
决堤：十九年不哭的人在他面前哭了。无声流泪，不擦不遮。龙尾从腰间松开，无力垂地
最深处的真话："我不想死。""我想回天山，我想和你一起回后山"
她没有说出的那句话。她说的是："……但是我回不去了。"
主角的回应：摘下她歪掉的凤翅金冠，放在地上，和匕首并排。"你回得去。我带你回去。"捡起"开山"递给她："拿着你的斧头。跟我走。"
场景十七：转机——故人的火
帐外中军火药库爆炸——非天山派所放
党项老将（场景十四中主动让路的守将）掀帐而入，单膝跪地："是我点的。"
老将身份：跟随废太子四十年的旧部，编入擒生军后隐忍至今。场景十四中看到天山派腰牌与"开山"后主动让路，就是在赌这一局
老将的安排：
火药库大火可烧至天亮，全营救火无暇他顾
提供党项女兵军服、令牌、羊皮地图（西北运粮小道 → 干河床 → 绕行回伊州）、巡逻换防间隙时间表
党项老将进来时扛了一个布袋，里面是流矢射杀的党项侍女尸体，年纪身量与钱塘君相近，面目已被烟熏模糊。火烧到大帐时，穿着郡主华服的尸体将被"发现"——军中无人见过"李钦若"真容，只认衣冠
老将决心以身入局，一起死在火中，消灭一切证据，并销毁一切可能暴露真相的痕迹
钱塘君换装：华服、金冠、玉带、耳坠逐一脱下穿在侍女尸体上。最后犹豫最久的是母亲的玉佩——最终挂在侍女脖子上。告别的不是侍女，是"李钦若"
老将转身走入火光。主角拉起钱塘君手腕："走。"

场景十八：撤离——只有两个人的路（上）
从大帐到伊州，只有他们两个人。所有同伴已在半个时辰前各自撤退。这段路程是全剧唯一的纯粹二人戏，也是他们关系的转折点——不是因为某一句惊天动地的话，而是因为所有伪装都已碎尽，他们第一次以真实的自己面对彼此，从共同的过去一路聊到彼此的心意。
18-A｜营区穿行
借火药库大火的混乱沿辎重通道向西北移动
钱塘君换上粗布军服，银白长发塞进军帽，龙尾缩在下摆里。主角帮她整理军帽时手指碰到她耳后，她微微缩了一下脖子但没有躲开
途中遭遇救火党项兵，躲入粮袋后方。短暂的、试图回归日常的对话（"你手上全是血" / "回去让玄天青看" / "玄天青的药比我做的肉干还难吃"），两人都愣住——太像平时的拌嘴，但谁都没有接下去
大营入口两名十七八岁的党项哨兵。钱塘君用西夏语伪装受惊女兵引开注意，空手点穴击昏，未杀。

18-B｜从沉默到开口
撤退的小道极度安静，只有两人脚步声和呼吸声
长久沉默后，钱塘君先开口。夜晚的黑暗成了天然的屏障，反而更容易说出口
话题一·开山步法（破冰）：
"你刚才……用开山九式的步法。" / "嗯。" / "什么时候学的。" / "没学。看你练的。" / "……看了多久。" / "十三年。"
身后传来一声极轻的、被死死咬住的抽泣。只有一声。他没有回头。她也不需要他回头
话题二·外堡往事（共同记忆的起点）：
又走了一段，她再次开口，声音平了一些："你还记不记得……外堡那条巷子。"
主角："记得。"
钱塘君："那时候我六岁。那些小孩朝我扔石子，喊我'党项杂种'。我不跑也不哭，就用龙尾把自己卷起来，把头埋在膝盖里。"（停顿）"然后你来了。"
主角："我也挨了一脸石子。"
钱塘君（声音里有一丝极淡的笑意）："你站在我前面，张开手挡着，跟他们说'再扔我就告诉破阵子长老'。你那时候才多大？比我还矮半个头。"
钱塘君："……那是第一次有人替我挡。"（声音轻了下去）"我那时候就想，这个人……以后我要对他好。"
主角："你天天拉我闯祸还让我背锅，那叫对我好？"
钱塘君："当然是对你好！"——语气里终于有了一点点平时的影子，但只有一点点，像废墟上试探性搭了一块砖

场景十九：撤离——只有两个人的路（下）
19-A｜干河床·星空——身世坦白
从粮道走到干涸河床，碎石遍地。月亮偏西，头顶是西域的星空，繁星密得像碎银洒在黑绒上
钱塘君停下脚步，转身面对主角。赤红色竖瞳在星光下亮得像两簇安静的火
"有件事……我想告诉你。真的。不是大帐里那些假话。"
她坦白了母亲的真相——不是"被俘受辱"，是真心相爱：
"我娘不是被俘的。她是自己留下来的。我爹……废太子，他救了她，后来他们相爱了。真的相爱。他给她写诗，西夏文的，叫她'天山的月亮'。"
"我爹想让两边不打仗。他学汉文化，他想和平。但是二皇子说他通敌叛国，怂恿皇帝废了他，杀了他全家。我是被一个老仆偷偷放出来的。"
"天山派找到我以后，他们不能说实话。天山的长老嫁给党项太子——哪怕是真心的——也是天山派的耻辱。所以他们说我娘是'被俘受辱'。我从小就知道真相，但我不能说。我说了，天山派就没脸了。"
"我活了十九年，每一天都在替别人撒谎。说我娘是受害者，说我爹是蛮贼，说我自己是耻辱。"（声音微颤）"你知道最难受的是什么吗？不是被人骂'党项杂种'。是我连替我爹娘喊一声冤都做不到。"
主角沉默了很久。然后他说："你爹娘的事……是他们的选择。不丢人。"
钱塘君愣住了。
主角："两个人真心相爱，不管是哪边的人，都不丢人。"
她低下头，银白长发垂下来遮住了脸。龙尾在身后轻轻晃了一下。很久之后，她说了一句极轻的："……谢谢你。"——这是她十九年来第一次听到有人这样说

19-B｜戈壁行进·月下——你把我当什么
两人继续沿干河床向伊州方向行进。四周空旷寂静，只有风声和脚步声
钱塘君走了一段，忽然问了一个她憋了很久的问题。声音很轻，像是怕问出口就再也收不回来：
"喂。我问你一件事。你得说实话。"
主角："你说。"
"你……把我当什么。"
沉默。
"我是说……从外堡那条巷子开始，这么多年相处。你到底把我当什么。"
（停顿，声音更小了）"当哥们儿？当师姐？当……一个需要你照顾的麻烦？"
她没有看他。她看着前方的戈壁，银白长发被夜风吹向一侧，露出侧脸的轮廓。龙尾在身后绷得笔直，尾尖微微颤抖——那是她极度紧张时才有的反应
主角没有立刻回答。他走了好几步才开口，声音不大，但很稳：
"外堡那天，我替你挡石子，不是因为你可怜。是因为你明明被砸得满头血，还是不哭。我觉得……这个人好厉害。"
"你问我把你当什么，我不知道该叫什么。但是你走了以后，我满天山找你。找不到的时候，我觉得天山都空了。那么大一座山，没有你在，就是空的。"
钱塘君停下了脚步。
她站在戈壁上，月光照着她的脸。赤红色竖瞳里的光芒在剧烈颤动，嘴唇抿成一条线，像是在拼命忍住什么。龙尾从绷直变成了轻轻颤抖，然后——啪地一下甩在地上，像是绷了太久的弦终于断了
她转过身面对他。深吸一口气。
"我也……不知道该叫什么。我问过姐姐，姐姐说我这叫'喜欢'。我说胡说八道，我和他是哥们。姐姐看了我一眼就没再说话了。"
"后来我自己想了很久。我想，如果这叫喜欢的话，那我好像……很早很早以前就开始了。早到我自己都不知道是什么时候开始的。"
（声音越来越小）"我不敢说。因为你对谁都好。我怕我说了，你只是……因为对谁都好，所以对我也好。那样的话我会很难受。"
"所以我就觉得，能和你以死党的关系相处也挺好的，我想……只要我一直在你身边，你总有一天也会看着我。"
主角看着她。月光下她的脸上有泪痕的痕迹，眼眶微红，但没有在哭。她只是站在那里，把藏了不知道多少年的话一句一句说出来，像是在清理一间堆满了杂物的房间，每搬出一件都轻松一点
他走上前一步。
"我会一直看着你。"
钱塘君的眼眶红了。她张了张嘴，想说什么，最后只是伸出手，轻轻缠上了他的手腕。像是怕弄疼他，又像是怕他消失
他没有躲。他反手握住了她的手。
两个人就这样站了一会儿。没有拥抱，没有亲吻，没有任何戏剧性的动作。只是手握着手，龙尾缠着手腕，在西域的星空下
这是他们在一起的第一刻。不是"哥们儿"，不是"师兄妹"，不是任何一种她用了十三年来伪装的关系。就是两个人，站在戈壁上，终于不再演了

19-C｜黎明前·展望未来
两人继续赶路，但气氛变了。不是沉重的变轻了，是多了一种从未有过的东西——像是在废墟上看到了第一棵绿芽
钱塘君的龙尾不再缩在军服下摆里了。它伸在外面，走路的时候会轻轻晃，走路的时候跟着步伐一左一右地摆。主角走在她旁边，两人之间的距离比之前近了半步——不是刻意靠近的，是自然而然地近了
钱塘君开始说一些以前从来不会说的话——不是沉重的，是轻的，是关于"以后"的：
"回去以后……你说破阵子会不会罚我？"
主角："肯定罚。"
"那你替我求情。"
主角："上次替你求情，我被罚抄了三百遍门规。"
钱塘君（理直气壮）："那你再抄三百遍。你是我什么人？替我挨罚不是应该的吗？"
——她说"你是我什么人"的时候，自己愣了一下。然后耳尖红了。龙尾啪地甩了一下地面，像是在掩饰什么。主角没有拆穿她，但嘴角动了一下
她开始认真地想"以后"——这是她很久很久没有做过的事，因为她曾经以为自己没有"以后"：
"回去以后，后山那个秘密基地得修一修。上次下雨塌了一半。"
"还有……我想去一趟天山北坡。师父以前说那边有一片野杏林，春天开花的时候特别好看。他说等我学完开山九式就带我去。后来他……没来得及。"（声音轻了一瞬，但很快又扬起来）"你陪我去。"
主角："好。"
钱塘君看了他一眼，赤红色竖瞳在星光下弯成了两道月牙。她没有说谢谢，也没有说任何煽情的话。她只是把龙尾从他手腕上松开，改成轻轻搭在他的肩膀上——很自然的动作，像是已经做过一千次一样
"还有一件事。"
主角："嗯？"
"以后……别对别人那么好了。"（声音忽然小了下去，语速变快，像是怕自己反悔）"就对我好就行了。多出来的好，本龙全部没收。"
"好。"

19-D｜伊州城墙·天亮了
天边泛起第一线灰白时，伊州城墙的轮廓出现在地平线上。城墙上守夜的火把微弱但确实亮着
钱塘君在看到城墙的那一刻停下了脚步。她站在戈壁上，看着那道她曾经站在上面杀人的城墙，看了很久
主角走到她旁边。两人并肩站着，看着远处的城墙和城墙后面更远处的天山雪峰——博格达峰主峰在晨光中泛着金色的光
"走啦。回家。"
她扛起"开山"，大步流星朝城门方向走去。军服下摆太长被绊了一下，龙尾甩出去保持平衡。晨光从天山方向照过来，把她的银白长发照得发亮，"开山"的斧刃在她肩头反射着金色光点
主角站在原地看了她一秒。然后迈开步子，跟了上去
两个人的影子被朝阳拉得很长很长，投在戈壁碎石上，一前一后，深深浅浅
天亮了

## 附录：叙事手法总结

| 手法 | 在本大纲中的运用 |
|------|-----------------|
| **信息差** | 全程主角是最后知道真相的人。钱塘君隐瞒所有人——包括洞庭君——独自行动。玩家通过她的独白和行为细节提前感知一切，主角浑然不觉。洞庭君作为最亲的人被排除在告别之外，比"知情者的沉默"更加残忍。 |
| **对比叙事** | 第三幕"最后的日常"是全剧最温暖的段落，紧接着就是最残忍的离别。第四幕大帐对峙中，身穿郡主华服的"李钦若"与平日嘻嘻哈哈的"钱塘君"形成极致反差；每一句冰冷的谎言后紧跟一段温暖的记忆闪回，言语与真相的对冲将张力拉到极致。 |
| **错位时空** | 主角在第四幕回忆起第三幕的每一个细节时，才意识到那些全是告别。"塔西·德力格尔"的真实含义、"如果本龙不在了"的试探——所有真相都是迟来的。 |
| **留白** | 没有诀别信，没有告别物品，没有任何直接的告别痕迹。钱塘君把"开山"留在原位，把房间整理得一切如常，唯一带走的只有母亲的玉佩——而这个细节只有洞庭君能读懂。主角是通过"床铺太整齐""玉佩不见了""寅时有人出城"这些碎片，一点一点拼凑出真相的。大帐对峙中，她的伪装同样是一种留白——她说了那么多谎话，唯独没有说出真心话；她的龙尾被华服遮住，她的颤抖被仪态压住，所有真实的情感都被藏在"没有说出口"的沉默里。 |
| **宏大碾压** | 三万铁鹞子为了一个十九岁的女孩围城。一场二十年前的宫廷政变，余波吞噬了一个从未享受过公主待遇的郡主。她的父亲因为主张和平而被杀，她因为父亲的血统而被追杀——而她甚至不能公开承认父母相爱的事实，因为这会让收养她的天山派蒙羞。历史的荒诞与残酷碾碎了所有个人的挣扎：她在城墙上杀的是父亲想要保护的族人，她在大帐中穿的是母亲当年的嫁衣，她用来推开主角的谎言，恰恰是天山派强加给她十九年的那个"官方版本"。她被迫用自己最痛恨的谎言，完成了最后的伪装。 |

## 附录二：关键意象与伏笔索引

| 意象/伏笔 | 首次出现 | 回收/爆发 | 含义 |
|-----------|---------|----------|------|
| **玉佩** | 场景三（深夜悬崖，钱塘君独自端详） | 场景九（洞庭君发现玉佩不见，确认钱塘君出走）；场景十四（守将说"她长得很像殿下"，暗示玉佩作为身份信物已被党项军确认） | 母亲的遗物，父母爱情的唯一实证，也是钱塘君身份的唯一铁证。她带走玉佩而非"开山"，说明她是以"李钦若"而非"钱塘君"的身份赴死的。 |
| **"开山"巨斧** | 贯穿全剧（钱塘君的标志性武器） | 场景九（主角发现"开山"还在房间，以为她没有离开）；场景十三（主角握着"开山"夜袭军营）；场景十五（主角将"开山"横在两人之间，要求她拿起斧头打赢他） | 她的武器，她的"命"。她把它留下，意味着她不打算活着回来。主角握着它去救她，意味着他在替她拿着她的"命"。最终在大帐中，这把斧头成为两人之间最后的纽带——他让她拿起斧头，本质上是在说"拿回你的命"。 |
| **"塔西·德力格尔"** | 场景七（最后的日常，她教主角说"外族话"） | 场景八（深夜独白中她说出真实含义）；场景十五（主角用这句话拆穿她的伪装） | 西夏语"请你平安"。她把告别藏在了一句主角听不懂的祝福里。这是她整个人物的缩影——用谎言包裹真心，用玩笑掩盖深情。当主角在大帐中说出这句话的真实含义时，她所有的伪装都失去了意义。 |
| **龙尾** | 贯穿全剧（钱塘君情绪的"泄密器"） | 场景二（看到告示时猛地绷直）；场景三（缠绕石柱越缠越紧）；场景七（说"哪儿也不去"时背后手攥拳，但龙尾的状态暗示真实情绪）；场景十五（被华服完全遮住——她连最后一个会泄露情绪的部分都藏起来了） | 龙尾是钱塘君无法完全控制的身体部位，是她真实情绪的外化。在大帐对峙中，她用华服遮住龙尾，象征着她对自我的彻底封锁——但这种"刻意遮掩"本身就是最大的破绽。 |
| **"哇哈哈哈"与"本龙"** | 贯穿全剧（钱塘君的标志性口癖） | 场景十五（整场对峙中完全消失，她以"李钦若"的端庄语气说话——直到最后举斧时，"本龙"二字回归） | "哇哈哈哈"和"本龙"是钱塘君的保护色，也是她十三年来构建的"自我"。在大帐中这些全部消失，意味着她在试图杀死"钱塘君"这个身份。但最后一刻"本龙"的回归，证明她杀不死。那个在雪地里打滚、把人按进雪堆、笑得前仰后合的女孩，才是真正的她。 |
| **"她长得很像殿下"** | 场景十四（守将放行时的一句话） | 待后续剧情回收（这句话暗示废太子与钱塘君母亲的关系并非"受辱"，而是有更深的渊源，为主角日后发现父母相爱的真相埋下伏笔） | 守将的这句话是一颗种子。主角此刻来不及细想，但这句话会留在他心里，成为日后追查真相的起点。当他最终发现钱塘君的父母是真心相爱时，他会想起这句话——"她长得很像殿下"——这不是一个仇人会说的话，这是一个臣子在怀念故主。 |
| **郡主华服** | 场景十五（钱塘君身穿赤金织锦长袍端坐大帐） | 场景十五全程（华服作为"李钦若"身份的视觉符号，与记忆闪回中的日常形象形成极致反差） | 华服是钱塘君的"铠甲"，也是她的"牢笼"。她穿上它，是为了让自己变成一个主角不认识的人，从而让主角放弃带她回去。但讽刺的是——这身华服是她母亲当年嫁入皇室时穿的，是父母爱情的见证。她用父母爱情的遗物，来完成对自己爱情的否定。 |

## 附录三：各角色在本剧情线中的定位与作用

| 角色 | 定位 | 关键作用 |
|------|------|---------|
| **钱塘君** | 核心悲剧主体 | 全部剧情的驱动力。她的隐藏身份是主要矛盾，她的自我牺牲是核心事件，她的伪装与真心的撕裂是情感张力的来源。 |
| **主角** | 信息差的承受者 / 真相的追寻者 | 全程处于信息盲区，是最后一个知道真相的人。他的"迟钝"不是因为愚蠢，而是因为钱塘君的伪装太过完美。他的觉醒和反击（大帐对峙中的逐条拆穿）是全剧的情感爆发点。 |
| **洞庭君** | 被排除在告别之外的至亲 | 她是钱塘君最亲的人，却被妹妹瞒过了。这种"被信任的人排除在告别之外"的痛苦，为悲剧增添了另一个维度。她在夜袭中的参与，是姐姐对妹妹最后的守护。 |
| **玄天青** | 真相的部分持有者 / 理性的声音 | 他知道钱塘君的身份（官方版本），在关键时刻向主角揭示真相。他的冷静与主角的冲动形成对比，但他的冷静背后也有愧疚——他没能阻止这一切。 |
| **破阵子** | 行动的发起者 / 师门的脊梁 | 从始至终主张"不交人"，在钱塘君出走后第一个拍桌说"追"。他代表天山派的热血与担当，也是夜袭行动的核心执行者。 |
| **苓雪妃** | 沉默的观察者 / 战场的支柱 | 她是最早察觉钱塘君异常的人（"她放过了一个党项小兵"）。在夜袭中以一人之力牵制铁鹞子亲卫，为主角创造窗口。她的沉默与行动形成强烈对比。 |
| **呼延显** | 日常的遮掩者 | 在城门口不动声色地挡住主角与告示，用玩世不恭的语气岔开话题。他代表"所有知情者都在替她隐瞒"这一信息差格局中的一环。 |
| **守将（党项老将）** | 历史的回声 | 他是废太子旧部，受过太子恩情。他的存在证明废太子并非"蛮贼"，而是一个有恩于人的仁者。他放行的举动和"她长得很像殿下"这句话，是父母爱情真相的第一道裂缝。 |
| **鹿椿若** | 对比叙事的工具 | 她本身没有与钱塘君的直接冲突，但主角照顾她的画面被钱塘君看到，强化了钱塘君"他对谁都好，对我也没什么特别"的错误判断，成为钱塘君决定独自赴死的心理推力之一。 |
