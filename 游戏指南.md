# 《姬侠传》游戏指南

> 一款基于 SillyTavern 前端的 LLM 互动武侠养成游戏

---

## 📖 游戏简介

《姬侠传》（独立前端版又名《瀚海》）是一款融合了养成、探索、战斗等多种玩法的武侠题材互动游戏。玩家将扮演一名初入天山派的新弟子，在门派中修炼武学、结交同门，并可下山闯荡江湖。游戏通过与 LLM（大语言模型）的交互，生成丰富多彩的剧情和故事。

游戏支持以下三种运行方式：
- **SillyTavern 版本**：搭配角色卡在 SillyTavern 中游玩，功能最完整
- **独立前端**：无需 SillyTavern，直接在浏览器中打开 HTML 文件游玩
- **Android APK**：打包为安卓应用，手机即可安装游玩

## 🔧 独立前端 / APK 连接指南

独立前端和 APK 版本需要手动配置 LLM API。在开局页面中点击「模型设置」按钮即可展开配置面板。

### LLM API 配置

| 项目 | 填写说明 |
|------|------|
| **API 类型** | OpenAI 兼容 或 Gemini，根据你的提供商选择 |
| **API 地址（Endpoint）** | 如 `https://api.openai.com/v1` 或中转站地址 |
| **API Key** | 你的密钥，仅保存在浏览器本地 |
| **模型选择** | 点「连接测试」后可从列表选择，或手动输入模型名 |
| **Temperature** | 推荐 0.8–1.0 |
| **最大输出 Token** | 建议 8000或者更大，过小可能会截断剧情 |
| **上下文窗口** | 建议匹配模型的实际上下文长度 |

**常用提供商地址参考：**

| 提供商 | API 地址 | API 类型 |
|--------|---------|--------|
| OpenAI 官方 | `https://api.openai.com/v1` | OpenAI 兼容 |
| Gemini 官方 | `https://generativelanguage.googleapis.com/v1beta` | Gemini |
| DeepSeek 官方 | `https://api.deepseek.com/v1` | OpenAI 兼容 |

> ⚙️ 本游戏已自带CORS 代理，相关选项维持默认设置即可。

### 向量化记忆（可选）

向量化是独立前端/APK 的独有功能，开启后每条历史摘要都会被向量化并存入，生成新消息时可语义匹配相关的历史信息。

| 项目 | 填写说明 |
|------|------|
| **启用向量召回** | 勾选就开启，需配置 Embedding 地址和 Key |
| **Embedding 地址** | 支持 OpenAI 兼容格式，如 `https://api.siliconflow.cn/v1` |
| **Embedding Key** | 独立的 API 密钥，不复用 LLM Key，建议用户自行注册硅基流动账号后免费获取 |
| **模型名称** | 推荐 `BAAI/bge-m3`（免费） |

**推荐免费 Embedding 提供商：**

| 提供商 | 地址 | 推荐模型 |
|--------|------|--------|
| SiliconFlow | `https://api.siliconflow.cn/v1` | `BAAI/bge-m3` |
| OpenAI | `https://api.openai.com/v1` | `text-embedding-3-small` |

> 💡 **提示**：完成配置后可先点「🔌 测试连接」验证，连接成功后点「🔄 重建索引」将现有历史全部向量化。历史存档越多，重建时间越长，耐心等待。

### 天赋属性

| 属性 | 影响 |
|------|------|

| **魅力** | 影响NPC好感度变化 ，影响正文中对主角相貌气质的描述|

### 人物数值

| 属性 | 说明 | 获取方式 |
|------|------|----------|
| **武学** | 武功造诣，影响战斗数值的获取 | 练武、切磋 |
| **学识** | 知识水平，影响可学习和可装备的武学 | 学习、切磋 |

### 战斗数值

| 属性 | 默认值 | 说明 |
|------|------|------|
| **攻击力** | 20 | 战斗伤害基数，加点+10 |
| **生命值** | 50 | 战斗血量上限，加点+25 |
| **暴击率** | 10% | 触发暴击的概率，暴击时按「暴击伤害」倍率造成伤害 |
| **暴击伤害** | 150% | 暴击时的伤害倍率，即暴击造成1.5倍伤害 |
| **格挡** | 0 | 每次受击固定减免的伤害（减后最低为1） |
| **穿甲** | 0 | 攻击时无视对方格挡的数量 |
| **回转** | 0 | 每次集气时额外回复的气 |
| **吸血** | 0 | 攻击命中时回复造成伤害的指定比例生命（%）|
| **反伤** | 0 | 受到伤害时对攻击方反弹等比例伤害（%）|

> 💡 **提示**：武学属性每提升一定数值会获得可分配点数，用于提升攻击力、生命值、暴击、格挡等多种属性，也通过装备和技能获得属性提升。

## 🏔️ 天山派地点与行动系统

游戏中的天山派包含以下可探索地点和对应的交互行动，每种行动都有对应的**影响天赋**和**获得属性**：
- **影响天赋**：该天赋越高，行动成功率越高
- **获得属性**：行动成功后提升的属性

| 地点 | 行动 | 影响天赋 | 获得属性 | 说明 |
|------|------|----------|----------|------|
| **演武场** | 练武 | 根骨 | 武学 | 修炼武功，提升武学造诣 |
| **藏经阁** | 学习 | 悟性 | 学识 | 阅读典籍，增长知识；此处还可学习战斗技能 |
| **伙房** | 打杂 | 根骨 | 金钱 | 帮厨干活，赚取报酬 |
| **伙房** | 交易 | — | — | 购买食物（胡饼、桂花糕、烤羊腿） |
| **后山** | 探索 | 悟性 | 金钱 | 探索山林，可能发现财物 |
| **后山** | 秘密赌场 | — | — | 21点小游戏，赢钱或输钱 |
| **议事厅** | 汇报 | 悟性 | 声望 | 向长老汇报，提升名声 |
| **铁匠铺** | 打铁 | 心性 | 金钱 | 锻造铁器，赚取工钱 |
| **铁匠铺** | 交易 | — | — | 购买装备 |
| **男弟子房** | 休息 | 心性 | 体力 | 休养生息，恢复体力 |
| **女弟子房** | 拜访 | 魅力 | 声望 | 拜访同门，提升人缘 |

| **公田** | 耕种 | — | — | 农场小游戏，种植获利 |
| **丹房** | 炼丹 | — | — | 炼丹小游戏，每周限一次 |

**操作方式**：点击地图上的建筑热区，会弹出地点信息面板，可选择"前往"进入该地点。

### 天赋与行动对照

根据你的天赋特长，可以选择更适合的行动：

| 天赋 | 擅长行动 | 建议 |
|------|----------|------|
| **根骨** | 练武、打杂 | 适合武学流和打工赚钱 |
| **悟性** | 学习、探索、汇报、炼丹 | 适合学识流和提升声望 |
| **心性** | 打铁、休息 | 适合稳健发展，心性还能免费行动 |
| **魅力** | 拜访 | 适合快速提升和NPC的好感度 |

### 行动成功率计算

行动的成功程度取决于以下公式：

```
最终值 = (随机数 + 天赋加成) × (体力 + 难度加成) / 100
```

其中：
- **随机数**：0-100的随机值
- **天赋加成**：对应天赋值（如练武时使用根骨值）
- **体力**：当前体力值（0-100）
- **难度加成**：简单+20，普通+10，困难+0

| 最终值 | 结果 | 体力消耗 | 属性获得 |
|--------|------|----------|----------|
| ≥80 | 大成功 | 0 | 高 |
| ≥60 | 成功 | 5 | 中 |
| ≥40 | 一般 | 10 | 低 |
| ≥20 | 失败 | 15 | 极低 |
| <20 | 大失败 | 20 | 无 |

### 心性特殊效果

心性天赋有一个特殊效果：**行动时有概率不消耗行动点**

- 概率 = 心性值 / 2（%）
- 例如：心性50时，有25%概率免费行动
- 触发时会显示「心性属性判定成功，本次行动不消耗行动点」

### 行动选择建议

1. **体力管理**：体力低时优先休息，体力影响所有行动成功率
2. **天赋匹配**：选择与自身高天赋匹配的行动，事半功倍
3. **小游戏例外**：农场、炼丹、赌场是小游戏，不受天赋直接影响

## ⏰ 时间与行动系统

### 时间单位

- **1年 = 48周**（12个月 × 4周）
- 每周有 **3个行动点**
- 季节随时间变化：春、夏、秋、冬

点击「跳过这一周」按钮可以：
- 时间推进一周
- 重置行动点为3
- 重置NPC送礼和切磋状态
- 可能触发特殊事件

## 👥 NPC系统

### NPC互动

1. **查看NPC**：进入场景后可看到在场的NPC
2. **与NPC互动**：点击NPC头像可选择「互动」或「切磋」
3. **送礼系统**：
   - 好感度≤40时可送礼
   - 消耗500金钱，提升5点好感度
   - 每周每个NPC只能送礼一次
4. **切磋**：与NPC进行回合制战斗，获胜后可获得相应奖励（每周每个NPC只能切磋一次）

### 好感度影响

根据游戏难度，好感度变化幅度不同：

| 难度 | 大幅上升 | 上升 | 下降 | 大幅下降 |
|------|----------|------|------|----------|
| 简单 | +4 | +2 | -1 | -2 |
| 普通 | +2 | +1 | -1 | -2 |
| 困难 | +2 | +1 | -2 | -4 |

### 魅力属性对好感度的影响

当NPC好感度发生 **正向变化**（上升/大幅上升）时，会触发魅力属性判定：

- **触发概率** = 魅力值 ÷ 2（%）
  - 例如：魅力50时，有 **25%** 概率触发
  - 魅力100时，有 **50%** 概率触发
- **判定效果**：好感度变化值 **翻倍**
- **判定提示**：触发时显示「对XXX的魅力属性判定成功，好感度变化加倍」

| 魅力值 | 触发概率 | 实际效果示例（普通难度「上升」）|
|--------|----------|--------------------------------|
| 20 | 10% | 10%概率+2，90%概率+1 |
| 40 | 20% | 20%概率+2，80%概率+1 |
| 60 | 30% | 30%概率+2，70%概率+1 |
| 80 | 40% | 40%概率+2，60%概率+1 |
| 100 | 50% | 50%概率+2，50%概率+1 |

> 💡 **注意**：魅力判定只对好感度上升生效，好感度下降时不触发

## ⚔️ 战斗系统

### 战斗类型

1. **切磋战斗**：与NPC友好比武
2. **事件战斗**：遭遇敌人时的战斗（可选择战斗或逃跑）

### 战斗流程

战斗采用回合制，在独立的战斗界面进行：
- 玩家属性基于「攻击力」和「生命值」
- 敌人属性分为「极低/低/中/高/极高」五档
- 每回合双方同时选择行动，然后结算
- 战斗胜利可获得奖励（金钱、道具等）

### 战斗属性

| 属性 | 说明 |
|------|------|
| **生命值** | 降至0则战斗失败/胜利 |
| **气** | 最大10点，用于释放攻击和绝招 |
| **基础伤害** | 攻击造成的基础伤害值 |

### 行动指令

战斗中可选择以下7种行动：

| 行动 | 快捷键 | 消耗气 | 效果说明 |
|------|--------|----------|----------|
| **集气** | Q | 0 | 气+1 |
| **防御** | W | 0 | 进入防御状态，大幅减少受到的伤害 |
| **轻攻击** | E | 1 | 造成100%基础伤害 |
| **重攻击** | R | 2 | 造成150%基础伤害，可无效化对方的轻攻击 |
| **绝招** | T | 5 | 造成300%基础伤害，无视防御，可无效化对方的轻攻击和重攻击 |
| **嘴炮** | Y | 0 | 挑衅对手让对手无法防御或无效化绝招 |
| **道具** | U | 0 | 使用战斗道具（每场限一次） |

### 行动克制关系

战斗存在复杂的行动克制体系：

```
防御   → 克制 → 普通攻击（大幅减少伤害）
重攻击 → 克制 → 轻攻击（无效化轻攻击并造成伤害）
绝招   → 克制 → 普通攻击（无效化轻/重攻击并造成伤害）
绝招   → 克制 → 防御（无视防御减伤）
嘴炮   → 克制 → 绝招（无效化绝招的伤害）
嘴炮   → 破防 → 防御（使对方气-1，进入挑衅状态）
```

### 场地效果

战斗开始时随机触发一种场地效果，持续 3 回合，对双方均生效：

| 场地效果 | 图标 | 效果 |
|----------|------|------|
| **杀意** | 🔥 | 双方攻击伤害×1.5，但每次攻击自己也受到造成伤害的25% |
| **瘴疠** | ☠️ | 双方每回合损失10%当前生命值 |
| **固守** | 🛡️ | 双方防御效果提升为完全免伤 |
| **气海** | ✨ | 双方每回合自动获得+1气 |
| **缠斗** | 🤝 | 双方无法使用嘴炮 |
| **回春** | 💚 | 双方每回合恢复10%最大生命值 |

点击战斗界面的场地效果指示器可查看详细说明。也可能不触发任何场地效果。

### 防御机制

| 被防御的攻击 | 受到伤害比例 |
|--------------|--------------|
| 轻攻击 | 15% |
| 重攻击 | 25% |
| 绝招 | 100%（无视防御） |

### 挑衅状态

- 被嘴炮破防后进入挑衅状态
- 持续 **2回合**
- 挑衅状态下 **无法使用防御**
- 能量-1（如果有能量的话）

### 战斗道具

每场战斗只能使用 **一次** 道具：

| 道具 | 效果 |
|------|------|
| **大力丸** | 本场战斗所有伤害×1.25 |
| **筋骨贴** | 本场战斗所受伤害÷1.25 |
| **金疮药** | 立即恢复30%最大生命值 |
| **霹雳丸** | 立即对敌人造成1.5倍攻击伤害 |

### 敌人战斗风格

敌人有四种战斗风格，影响其AI行为倾向：

| 风格 | 特点 | 名字颜色 |
|------|------|----------|
| **风** | 快攻型，偏好轻攻击和嘴炮 | 靛蓝色 |
| **火** | 爆发型，偏好重攻击 | 暗红色 |
| **林** | 蓄力型，偏好集气和绝招 | 深绿色 |
| **山** | 防守型，偏好防御 | 土黄色 |

### 战斗快捷键

| 按键 | 功能 |
|------|------|
| Q/W/E/R/T/Y/U | 选择对应行动 |
| Enter | 确认行动 |
| ESC | 取消选择/关闭弹窗 |

## 技能系统

### 技能基础

- 在**藏经阁**可找到「技能习得」按钮，进入技能学习界面
- 在**属性查看**下拉菜单可找到「技能管理」按钮，调整技能装备
- 技能分为 **攻击 / 防御 / 辅助 / 控制** 四大类，每个技能有 1~5 级
- 升级技能需要消耗**金钱**，并满足前置条件（特定属性值或NPC好感度）

### 记忆点限制

装备技能需要消耗「记忆点」：

- **最大记忆点** = 学识 / 20（向下取整）
- 每个技能按等级占用不同记忆点数（1~3点）
- 超过记忆上限时无法装备新技能，需先卸下已装备的技能
- 学识越高，可同时装备的技能越多、越强

## 🎲 小游戏系统

### 秘密赌场（21点）

- 位于：后山
- 玩法：经典21点扑克赌博游戏
- 可赢取或输掉金钱

#### 游戏目标

使手牌点数尽可能接近 **21点**，但不能超过21点（爆牌）。点数比庄家高即获胜。

#### 点数计算

| 牌面 | 点数 |
|------|------|
| 2-10 | 面值（2=2点，10=10点） |
| J / Q / K | 10点 |
| A（Ace） | 11点或1点（自动最优选择） |

> 💡 **A的计算**：系统会自动选择最优点数。如果算11点会爆牌，则自动算作1点。

#### 游戏流程

1. **押注**：输入押注金额（不能超过总金额）
2. **发牌**：点击「开始游戏」，庄家和玩家各发2张牌
3. **玩家回合**：
   - 点击「要牌」：再抽一张牌
   - 点击「停牌」：结束要牌，进入庄家回合
   - 如果点数超过21点，立即爆牌失败
4. **庄家回合**：庄家自动按规则要牌（点数<17必须要牌）
5. **结算**：比较双方点数，决定胜负

#### 胜负判定

| 情况 | 结果 | 金钱变化 |
|------|------|----------|
| 玩家爆牌（>21点） | 庄家赢 | 损失押注金额 |
| 庄家爆牌（>21点） | 玩家赢 | 获得押注金额 |
| 玩家点数 > 庄家点数 | 玩家赢 | 获得押注金额 |
| 玩家点数 = 庄家点数 | 平局 | 押注金额返还 |
| 玩家点数 < 庄家点数 | 庄家赢 | 损失押注金额 |

#### 操作说明

| 按钮 | 功能 |
|------|------|
| **开始游戏** | 确认押注并发牌 |
| **要牌** | 再抽一张牌 |
| **停牌** | 结束要牌，让庄家行动 |
| **×（左上角）** | 退出游戏，金钱同步回主游戏 |

> ⚠️ **注意**：金钱归零会触发破产，自动退出游戏！

- 位于：公田
- 玩法：种植作物获取收益

#### 农场基础

- **农场规模**：4×3 网格，共12块田地
- **交互限制**：每块田地每周只能进行一次操作（浇水/除虫/施肥三选一）

#### 作物种类

| 作物 | 成熟周期 | 种子价格 | 基础售价 | 满品质售价 |

> 💡 **售价公式**：最终售价 = 基础售价 × (1 + 品质/100)，品质上限100

#### 作物属性

#### 农场操作

| 操作 | 效果 | 消耗 | 备注 |

| **浇水** | 成熟度+1 | 无 | 每周每地一次 |
| **除虫** | 健康值+30~50 | 无 | 每周每地一次 |
| **施肥** | 品质+10~30 | $5 | 每周每地一次 |

- 健康值降至 **0** 的作物死亡
- 成熟后超过 **5周** 未收割会腐烂消失
