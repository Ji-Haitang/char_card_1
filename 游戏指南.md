# 《姬侠传》游戏指南

> 一款基于 SillyTavern 前端的 LLM 互动武侠养成游戏

---

## 📖 游戏简介

《姬侠传》是一款融合了养成、探索、战斗等多种玩法的武侠题材互动游戏。玩家将扮演一名初入天山派的新弟子，在门派中修炼武学、结交同门，并可下山闯荡江湖。游戏通过与 LLM（大语言模型）的交互，生成丰富多彩的剧情和故事。

## 🏔️ 天山派地点与行动系统

- **影响天赋**：该天赋越高，行动成功率越高
- **获得属性**：行动成功后提升的属性

| **藏经阁** | 学习 | 悟性 | 学识 | 阅读典籍，增长知识 |
| **伙房** | 打杂 | 根骨 | 金钱 | 帮厨干活，赚取报酬 |
| **伙房** | 交易 | — | — | 购买食物（胡饼、桂花糕、烤羊腿） |
| **后山** | 探索 | 悟性 | 金钱 | 探索山林，可能发现财物 |
| **后山** | 秘密赌场 | — | — | 21点小游戏，赢钱或输钱 |
| **议事厅** | 汇报 | 悟性 | 声望 | 向长老汇报，提升名声 |
| **铁匠铺** | 打铁 | 心性 | 金钱 | 锻造铁器，赚取工钱 |
| **铁匠铺** | 交易 | — | — | 购买装备 |

**操作方式**：点击地图上的建筑热区，会弹出地点信息面板，可选择"前往"进入该地点。

### 天赋与行动对照

根据你的天赋特长，可以选择更适合的行动：

| 天赋 | 擅长行动 | 建议 |
|------|----------|------|
| **根骨** | 练武、打杂 | 适合武学流和打工赚钱 |
| **悟性** | 学习、探索、汇报、炼丹 | 适合学识流和提升声望 |
| **心性** | 打铁、休息 | 适合稳健发展，心性还能免费行动 |
| **魅力** | 拜访 | 适合快速提升和NPC的好感度 |

### 行动成功率计算

行动的成功程度取决于以下公式：

```
最终值 = (随机数 + 天赋加成) × (体力 + 难度加成) / 100
```

其中：
- **随机数**：0-100的随机值
- **天赋加成**：对应天赋值（如练武时使用根骨值）
- **体力**：当前体力值（0-100）
- **难度加成**：简单+20，普通+10，困难+0

| 最终值 | 结果 | 体力消耗 | 属性获得 |
|--------|------|----------|----------|
| ≥80 | 大成功 | 0 | 高 |
| ≥60 | 成功 | 5 | 中 |
| ≥40 | 一般 | 10 | 低 |
| ≥20 | 失败 | 15 | 极低 |
| <20 | 大失败 | 20 | 无 |

### 心性特殊效果

心性天赋有一个特殊效果：**行动时有概率不消耗行动点**

- 概率 = 心性值 / 2（%）
- 例如：心性50时，有25%概率免费行动
- 触发时会显示「心性属性判定成功，本次行动不消耗行动点」

### 行动选择建议

1. **体力管理**：体力低时优先休息，体力影响所有行动成功率
2. **天赋匹配**：选择与自身高天赋匹配的行动，事半功倍
3. **小游戏例外**：农场、炼丹、赌场是小游戏，不受天赋直接影响

## 👥 NPC系统

### NPC互动

1. **查看NPC**：进入场景后可看到在场的NPC
2. **与NPC互动**：点击NPC头像可选择「互动」或「切磋」
3. **送礼系统**：
   - 好感度≤40时可送礼
   - 消耗500金钱，提升5点好感度
   - 每周每个NPC只能送礼一次
4. **切磋**：与NPC进行回合制战斗，获胜后可获得相应奖励（每周每个NPC只能切磋一次）

### 好感度影响

根据游戏难度，好感度变化幅度不同：

| 难度 | 大幅上升 | 上升 | 下降 | 大幅下降 |
|------|----------|------|------|----------|
| 简单 | +4 | +2 | -1 | -2 |
| 普通 | +2 | +1 | -1 | -2 |
| 困难 | +2 | +1 | -2 | -4 |

### 魅力属性对好感度的影响

当NPC好感度发生 **正向变化**（上升/大幅上升）时，会触发魅力属性判定：

- **触发概率** = 魅力值 ÷ 2（%）
  - 例如：魅力50时，有 **25%** 概率触发
  - 魅力100时，有 **50%** 概率触发
- **判定效果**：好感度变化值 **翻倍**
- **判定提示**：触发时显示「对XXX的魅力属性判定成功，好感度变化加倍」

| 魅力值 | 触发概率 | 实际效果示例（普通难度「上升」）|
|--------|----------|--------------------------------|
| 20 | 10% | 10%概率+2，90%概率+1 |
| 40 | 20% | 20%概率+2，80%概率+1 |
| 60 | 30% | 30%概率+2，70%概率+1 |
| 80 | 40% | 40%概率+2，60%概率+1 |
| 100 | 50% | 50%概率+2，50%概率+1 |

> 💡 **注意**：魅力判定只对好感度上升生效，好感度下降时不触发

## ⚔️ 战斗系统

### 战斗类型

1. **切磋战斗**：与NPC友好比武
2. **事件战斗**：遭遇敌人时的战斗（可选择战斗或逃跑）

### 战斗流程

战斗采用回合制，在独立的战斗界面进行：
- 玩家属性基于「攻击力」和「生命值」
- 敌人属性分为「极低/低/中/高/极高」五档
- 每回合双方同时选择行动，然后结算
- 战斗胜利可获得奖励（金钱、道具等）

### 战斗属性

| 属性 | 说明 |
|------|------|
| **生命值** | 降至0则战斗失败/胜利 |
| **气** | 最大10点，用于释放攻击和绝招 |
| **基础伤害** | 攻击造成的基础伤害值 |

### 行动指令

战斗中可选择以下7种行动：

| 行动 | 快捷键 | 消耗气 | 效果说明 |
|------|--------|----------|----------|
| **集气** | Q | 0 | 气+1 |
| **防御** | W | 0 | 进入防御状态，大幅减少受到的伤害 |
| **轻攻击** | E | 1 | 造成100%基础伤害 |
| **重攻击** | R | 2 | 造成150%基础伤害，可无效化对方的轻攻击 |
| **绝招** | T | 5 | 造成300%基础伤害，无视防御，可无效化对方的轻攻击和重攻击 |
| **嘴炮** | Y | 0 | 挑衅对手让对手无法防御或无效化绝招 |
| **道具** | U | 0 | 使用战斗道具（每场限一次） |

### 行动克制关系

战斗存在复杂的行动克制体系：

```
防御 → 克制 → 普通攻击（大幅减少伤害）
重攻击 → 克制 → 轻攻击（无效化轻攻击并造成伤害）
绝招   → 克制 → 普通攻击（无效化轻/重攻击并造成伤害）
绝招   → 克制 → 防御（无视防御减伤）
嘴炮   → 克制 → 绝招（无效化绝招的伤害）
嘴炮   → 破防 → 防御（使对方气-1，进入挑衅状态）
```

### 防御机制

| 被防御的攻击 | 受到伤害比例 |
|--------------|--------------|
| 轻攻击 | 15% |
| 重攻击 | 25% |
| 绝招 | 100%（无视防御） |

### 挑衅状态

- 被嘴炮破防后进入挑衅状态
- 持续 **2回合**
- 挑衅状态下 **无法使用防御**
- 能量-1（如果有能量的话）

### 战斗道具

每场战斗只能使用 **一次** 道具：

| 道具 | 效果 |
|------|------|
| **大力丸** | 本场战斗所有伤害×1.25 |
| **筋骨贴** | 本场战斗所受伤害÷1.25 |
| **金疮药** | 立即恢复30%最大生命值 |
| **霹雳丸** | 立即对敌人造成1.5倍攻击伤害 |

### 敌人战斗风格

敌人有四种战斗风格，影响其AI行为倾向：

| 风格 | 特点 | 名字颜色 |
|------|------|----------|
| **风** | 快攻型，偏好轻攻击和嘴炮 | 靛蓝色 |
| **火** | 爆发型，偏好重攻击 | 暗红色 |
| **林** | 蓄力型，偏好集气和绝招 | 深绿色 |
| **山** | 防守型，偏好防御 | 土黄色 |

### 战斗快捷键

| 按键 | 功能 |
|------|------|
| Q/W/E/R/T/Y/U | 选择对应行动 |
| Enter | 确认行动 |
| ESC | 取消选择/关闭弹窗 |

## 🎲 小游戏系统

### 秘密赌场（21点）

- 位于：后山
- 玩法：经典21点扑克赌博游戏
- 可赢取或输掉金钱

#### 游戏目标

使手牌点数尽可能接近 **21点**，但不能超过21点（爆牌）。点数比庄家高即获胜。

#### 点数计算

| 牌面 | 点数 |
|------|------|
| 2-10 | 面值（2=2点，10=10点） |
| J / Q / K | 10点 |
| A（Ace） | 11点或1点（自动最优选择） |

> 💡 **A的计算**：系统会自动选择最优点数。如果算11点会爆牌，则自动算作1点。

#### 游戏流程

1. **押注**：输入押注金额（不能超过总金额）
2. **发牌**：点击「开始游戏」，庄家和玩家各发2张牌
3. **玩家回合**：
   - 点击「要牌」：再抽一张牌
   - 点击「停牌」：结束要牌，进入庄家回合
   - 如果点数超过21点，立即爆牌失败
4. **庄家回合**：庄家自动按规则要牌（点数<17必须要牌）
5. **结算**：比较双方点数，决定胜负

#### 胜负判定

| 情况 | 结果 | 金钱变化 |
|------|------|----------|
| 玩家爆牌（>21点） | 庄家赢 | 损失押注金额 |
| 庄家爆牌（>21点） | 玩家赢 | 获得押注金额 |
| 玩家点数 > 庄家点数 | 玩家赢 | 获得押注金额 |
| 玩家点数 = 庄家点数 | 平局 | 押注金额返还 |
| 玩家点数 < 庄家点数 | 庄家赢 | 损失押注金额 |

#### 操作说明

| 按钮 | 功能 |
|------|------|
| **开始游戏** | 确认押注并发牌 |
| **要牌** | 再抽一张牌 |
| **停牌** | 结束要牌，让庄家行动 |
| **×（左上角）** | 退出游戏，金钱同步回主游戏 |

> ⚠️ **注意**：金钱归零会触发破产，自动退出游戏！

### 农场系统

- 位于：公田
- 玩法：种植作物获取收益

#### 农场基础

- **农场规模**：4×3 网格，共12块田地
- **交互限制**：每块田地每周只能进行一次操作（浇水/除虫/施肥三选一）

#### 作物种类

| 作物 | 成熟周期 | 种子价格 | 基础售价 | 满品质售价 |
|------|----------|----------|----------|------------|
| 小麦 | 6周 | $75 | $360 | $720 |
| 茄子 | 8周 | $115 | $560 | $1120 |
| 甜瓜 | 10周 | $165 | $825 | $1650 |
| 甘蔗 | 12周 | $240 | $1200 | $2400 |

> 💡 **售价公式**：最终售价 = 基础售价 × (1 + 品质/100)，品质上限100

#### 作物属性

| 属性 | 范围 | 说明 |
|------|------|------|
| **成熟度** | 0~需求值 | 达到作物要求才能收割 |
| **健康值** | 0~100 | 低于50无法收割，降至0作物死亡 |
| **品质** | 0~100+ | 影响最终售价，上限按100计算 |

#### 农场操作

| 操作 | 效果 | 消耗 | 备注 |
|------|------|------|------|
| **种植** | 在空地种下种子 | 1颗种子 | 需先从商店购买种子 |
| **浇水** | 成熟度+1 | 无 | 每周每地一次 |
| **除虫** | 健康值+30~50 | 无 | 每周每地一次 |
| **施肥** | 品质+10~30 | $5 | 每周每地一次 |
| **收割** | 获得金钱 | 消耗作物 | 需成熟且健康≥50 |

#### 批量操作

通过顶部「统一处理」下拉菜单可批量操作：

| 操作 | 效果 | 条件 |
|------|------|------|
| **统一浇水** | 所有作物成熟度+1 | 本周未操作的作物 |
| **统一除虫** | 所有作物健康+30~50 | 健康<50且未操作的作物 |
| **统一施肥** | 所有作物品质+10~30 | 本周未操作的作物，每块$5 |

#### 每周自然变化

每当主游戏时间推进一周，农场自动发生：

- 所有作物成熟度 **+1**
- 所有作物健康值 **-10~20**（随机）
- 健康值降至 **0** 的作物死亡
- 成熟后超过 **5周** 未收割会腐烂消失

#### 随机事件

每周有概率触发随机事件：

| 事件 | 概率 | 效果 |
|------|------|------|
| 🌧️ **雨水充沛** | 20% | 所有作物成熟度额外+1 |
| 🦗 **蝗虫过境** | 10% | 所有作物健康值额外-30 |

### 炼丹系统

- 位于：丹房
- 玩法：炼制丹药（每周限一次）

#### 炼丹基础

- **药材需求**：需投入 **6-10个** 药材才能开始炼丹
- **操作次数**：每次炼丹有 **15次** 操作机会
- **初始品质**：= (投入药材数 - 6) × 5
  - 6个药材 → 初始品质0
  - 10个药材 → 初始品质20

#### 药材种类

| 药材 | 对应属性 | 购买价格 |
|------|----------|----------|
| 丹参 | 根骨 | 500金 |
| 当归 | 悟性 | 500金 |
| 没药 | 心性 | 500金 |
| 沉香 | 魅力 | 500金 |

> 💡 炼丹成功后，会根据投入的药材比例随机提升对应属性

#### 核心数值

| 数值 | 范围 | 说明 |
|------|------|------|
| **炉温** | 0~100+ | 超过100会炸炉失败 |
| **品质** | 0~100 | 决定最终产出等级 |
| **完成度** | 0~100 | 达到100即炼丹完成 |

#### 炉温倍率

炉温倍率影响品质和完成度的增长速度：

| 炉温范围 | 倍率范围 | 说明 |
|----------|----------|------|
| 0~30 | 0.3~0.5x | 低效区 |
| 31~70 | 0.5~1.0x | 稳定区 |
| 71~100 | 1.0~1.8x | 高效但危险 |

#### 操作指令

| 操作 | 炉温变化 | 品质变化 | 完成度变化 | 特点 |
|------|----------|----------|------------|------|
| **猛火** | +25~35 | -5~-2 | 0 | 快速升温，略降品质 |
| **文火** | +5~10 | +3~6 × 倍率 | +4~8 × 倍率 | 微调温度，稳定提升 |
| **投药** | -15~-5 | +2~4 × 倍率 | +8~15 × 倍率 | 降温，大幅提升完成度 |
| **冷凝** | -30~-20 | +8~12 × 倍率 | +0~2 × 倍率 | 大幅降温，提升品质 |
| **淬炼** | +5~10 | +5~8 × 倍率 | -10~-5 × 倍率 | 提升品质，降低完成度 |

#### 丹炉状态

| 炉温 | 状态 | 说明 |
|------|------|------|
| <30 | 冷寂 | 效率较低 |
| 30~70 | 正常 | 最佳操作区间 |
| 71~90 | 炽热 | 高效但需注意降温 |
| >90 | 将炸 | 危险！需立即冷凝 |
| >100 | 炸炉 | 炼丹失败 |

#### 炼丹产出

根据完成时的品质决定产出：

| 品质 | 产出 | 属性提升 |
|------|------|----------|
| 100 | 九转金丹（传说） | 随机属性 **+3** |
| 90~99 | 易筋丹（史诗） | 随机属性 **+2** |
| 75~89 | 培元丹（稀有） | 随机属性 **+1** |
| 50~74 | 普通丹药（随机） | 无 |
| <50 | 无产出 | 无 |

普通丹药随机获得以下之一：大力丸、筋骨贴、金疮药、霹雳丸

#### 随机事件

炼丹过程中可能触发随机事件：

| 类型 | 事件 | 效果 |
|------|------|------|
| **负面** | ❄️寒潮 | 炉温-15，下次猛火减半 |
| **负面** | 🚫淤积 | 下次冷凝失效 |
| **负面** | 🔥反噬 | 每轮品质-5（持续3轮） |
| **负面** | 😵心魔 | 下次波动翻倍 |
| **正面** | ✨祥瑞 | 品质+15 |
| **正面** | 💡顿悟 | 下次不耗操作次数 |
| **正面** | 🌪️风助 | 下次猛火效果×2.5 |
| **正面** | 🛡️护主 | 抵挡一次炸炉 |
| **中性** | ♨️沸腾 | 所有效果翻倍（持续2轮） |
| **中性** | 🔄阴火 | 下次温度变化反转 |
| **中性** | 💎凝丹 | 倍率锁定1.5x（持续2轮） |

#### 炼丹策略建议

1. **控温优先**：保持炉温在40-70之间，兼顾效率和安全
2. **冷凝救命**：炉温接近90时优先使用冷凝
3. **品质优先**：完成度够用时多用淬炼提升品质
4. **关注事件**：正面事件可大幅提升效率，负面事件需及时应对
5. **药材配比**：根据想提升的属性选择对应药材
