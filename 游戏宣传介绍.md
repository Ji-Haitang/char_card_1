北宋初年，党项崛起，陇右局势风云变幻。

## 主要玩点是什么？

### 🖥️ 全图形界面，移动端同样好用
不是纯文字菜单。门派是可以点击的地图，每栋建筑都是一个交互场景；下山探索有完整的世界地图；战斗有独立战斗界面和角色立绘差分动画。  
**安卓 APK 可直接安装，手机躺着玩不是梦。**

### 👥 10+ 可攻略角色，兽耳娘大集合
傲娇师妹、清冷师尊、合法1011、元气女兄弟、文学少女、千金大小姐……  
狐娘、猫娘、龙娘、兔娘——堪称武侠版明日方舟。  
好感度实时驱动 NPC 的态度和语气，从冷淡疏离到芳心暗许，**每个阶段都有反馈**。 
全部可攻略角色附带NSFW行为指导，每个角色都有特殊设定，瑟瑟体验告别千篇一律 
有男不玩？自主勾选出场角色，打造你的专属阵容。  

### ⚔️ 自主操作绝非摆设的战斗系统
- **7 种战斗指令**，防御/轻攻/重攻/绝招/嘴炮/集气/道具，存在完整克制链
- **4 种敌人战斗风格**：风（快攻）、火（爆发）、林（蓄力）、山（防守）
- **16 个战斗技能**，每个技能的解锁前提都和特定 NPC 好感度绑定，强度靠刷好感
- **暴击/穿甲/格挡/回转/吸血/反伤** 等二级属性，装备真的有用
- **6 种随机场地效果**（杀意/瘴疠/固守/气海/缠斗/回春）
- 战利品掉落、战斗道具、角色战斗立绘差分演出

### 🎒 武学系统与严格数值
武学成长决定战斗属性的获取，学识决定可装备技能的上限。  
属性点分配、装备词条、技能触发——数值严格结算，AI **只负责演绎结果，不负责判定**。  
结算前端做，口胡 AI 改不了。

### 🗺️ SLG 门派养成 + GAL 大地图探索，双模式切换
**SLG 模式**：演武场练武、藏经阁习技、铁匠铺打铁、伙房打杂、后山探索……每周 3 点行动点，时间管理是门学问。昼夜四季流转，NPC 有自己的时间表。  
**GAL 模式**：十数个详细设定的西域地点，高昌城、月牙泉、千佛洞、魔教总坛……危险度分级，随机事件与战斗事件并存，好感度达标可带心仪的 TA 同行，触发专属剧情和 NSFW CG。

### 🎰 不只是聊天——四种玩法小游戏
- **秘密赌场**：21 点博弈，一夜暴富或倾家荡产
- **农场种植**：耕耘田地、4 种作物、种田也能致富
- **炼丹系统**：炉温倍率机制，随机事件有正有负，九转金丹 or 炸炉？
- **切磋对战**：和所有 NPC 都能切磋，胜负有奖励，是刷好感的重要手段

### 🎵 BGM系统
每个场景、不同剧情氛围都有对应背景音乐，设置中可独立开关BGM并调节音量。特殊剧情段落会自动切换特殊BGM。

### 🧠 不让 AI 写废话的提示词工程
内置**反八股文风**提示词——禁止"不是 X 而是 Y"式对比转折、禁止意义空洞强行升华的总结收尾，见微知著，引导 AI 用动作和细节推进故事。  
**自动历史总结**：两档压缩量级，自动提炼剧情精华，突破 token 上限，长线剧情不断档。  
**向量化召回**：历史记录向量化入库，生成新回复时语义匹配最相关历史内容优先注入上下文，让AI自动记住过去的事件细节。

### 💾 多种存档方式，兼容酒馆
- 每次 LLM 回复后自动保存，最多保留 3 个滚动存档。
- 导出 JSON 格式存档文件。
- 支持酒馆对话log作为存档导入

## 怎么开始玩？

1. **下载 APK** 安装到安卓手机，或直接**浏览器打开** 游玩地址
2. 点击「**模型设置**」，填入你的 LLM API 地址和 Key（支持 OpenAI 兼容格式和 Gemini）
3. 开始新游戏，自定义角色名和出身，分配初始属性点
4. 开玩

> 推荐模型：任何支持长上下文的强力指令模型均可。中文生成建议 DeepSeek V4 pro / Gemini 3.5 flash / Claude 系列。  
> 需要向量化记忆 API？[SiliconFlow](https://api.siliconflow.cn) 免费注册，填入key开箱即用。
